# LLApp
## Приложение для изучения иностранных языков
### Приложение было разработано для следующих групп пользователей:
1. люди, изучающие иностранные языки для повышения своих навыков 
коммуникации;
2. люди, изучающие иностранные языки для профессионального роста;
3. ученики и студенты, которым необходимо изучать иностранные языки в рамках 
учебной программы;
4. путешественники, желающие овладеть базовыми знаниями языка перед 
поездкой в другую страну. 

### Цель 
Цель - предоставить пользователям эффективный инструмент для изучения иностранных 
языков, который поможет им развивать навыки говорения, понимания, чтения и письма.

### Задачи
1. предоставление обучающего контента в виде словарей;
2. предоставление возможности изучать несколько языков;
3. предоставление обучающего контента в виде грамматических правил;
4. предоставление возможности проверять свои знания с помощью тестов.
